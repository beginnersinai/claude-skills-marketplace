# Beginners in AI — Claude Skills Marketplace

Free Claude skills and plugins built by [Beginners in AI](https://beginnersinai.org). Research-backed, tested on real businesses.

## How to Install

### Add the marketplace (one time):
In Claude Code terminal:
```
/plugin marketplace add beginnersinai/claude-skills-marketplace
```

### Then install any plugin:
```
/plugin install ai-mapping
```

## Available Plugins

| Plugin | Description | Based On |
|--------|-------------|----------|
| **ai-mapping** | Find where AI should be in your business but isn't. Scans 10 core functions, scores opportunities, generates an action plan. | Harvard/INSEAD research (515 startups, 44% more AI use cases, 1.9x higher revenue) |
